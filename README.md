# Primer API REST con Express.js

Este proyecto consiste en una **API RESTful** básica desarrollada con **Node.js** y **Express.js** para gestionar y consultar información de alumnos de forma rápida.

---

## 📂 Estructura del Proyecto

En el explorador de archivos se pueden identificar los componentes principales del entorno:
* **`index.js`**: El punto de entrada de la aplicación donde se inicializa el servidor Express y se programan las rutas.
* **`package.json`**: Archivo de configuración que gestiona las dependencias del proyecto (como Express).
* **`Lista de Estudiantes.xlsx`**: Un archivo de Excel que se encuentra en la raíz, utilizado habitualmente como base de datos local o fuente de datos para el sistema.

---

Para verificar el funcionamiento, se realiza una petición HTTP en entorno local utilizando la extensión de pruebas integrada:
